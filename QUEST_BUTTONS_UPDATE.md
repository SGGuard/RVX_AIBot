# 🎯 Интерактивные кнопки квестов - Обновление v0.12.1

**Дата**: 30 ноября 2025  
**Версия**: v0.12.1  
**Изменение**: Замена текстовых команд на интерактивные кнопки

## ✨ Что изменилось

### Было (v0.12.0) ❌
```
👉 Команда: /quest_what_is_dex
👉 Команда: /quest_what_is_staking
```
Пользователю приходилось вводить команду вручную.

### Стало (v0.12.1) ✅
```
[▶️ Начать: Что такое DEX?]
[▶️ Начать: Что такое стейкинг?]
```
Одно нажатие - и квест начинается!

## 🔧 Технические изменения

### tasks_command() (bot.py)
- Теперь строит клавиатуру с интерактивными кнопками
- Каждый квест имеет кнопку "▶️ Начать: [название]"
- Callback data: `start_quest_{quest_id}`

### button_callback() (bot.py)
- Добавлена проверка на `start_quest_*` callbacks
- Парсит quest_id из callback_data
- Вызывает `start_quest()` при нажатии кнопки

## 🔗 Flow

```
User видит /tasks
    ↓
Видит список квестов с кнопками
    ↓
Нажимает [▶️ Начать: Что такое DEX?]
    ↓
Отправляется callback: start_quest_what_is_dex
    ↓
button_callback() обрабатывает callback
    ↓
start_quest() запускается с quest_id="what_is_dex"
    ↓
Показывается материал + тест
```
