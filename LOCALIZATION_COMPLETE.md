### ✅ 4. Обработчики в bot.py

#### a) start_command (строка 6136)
- Проверяет наличие языка у пользователя
- Если нет - показывает меню выбора
- Если есть - переходит к главному меню

#### b) handle_language_selection (строка 9670)
- Обрабатывает callback_data="lang_ru" и "lang_uk"
- Сохраняет язык в БД через `set_user_language()`
- Показывает главное меню на выбранном языке

#### c) button_callback (добавлен обработчик settings_menu)
- Новый обработчик для data="settings_menu"
- Позволяет переключать язык в настройках
- Показывает меню выбора языка

#### d) Проверка подписки + язык
- При проверке подписки также предлагает выбрать язык
- Интегрирован в check_subscription обработчик

## 🚀 Использование В Коде

### Получить текст на языке пользователя:
```python
from i18n import get_text

# Способ 1: Используя user_id (будет найден язык из БД)
text = await get_text("start.greeting", user_id)

# Способ 2: Указав язык явно
text = await get_text("start.greeting", language="uk")

# Способ 3: С параметрами форматирования
text = await get_text("start.greeting", user_id, name="Ivan")
```

### Установить/получить язык:
```python
from i18n import set_user_language, get_user_language

# Установить язык
await set_user_language(user_id, "uk")

# Получить язык
lang = get_user_language(user_id)  # вернет "uk" или "ru"
```

### В обработчиках Telegram:
```python
async def my_handler(update: Update, context: ContextTypes.DEFAULT_TYPE):
    user_id = update.effective_user.id
    
    # Текст автоматически получится на языке пользователя
    greeting = await get_text("start.greeting", user_id)
    await update.message.reply_text(greeting)
```

## 🔄 Процесс Смены Языка

1. **В меню:** Settings → Language Selection
2. **Callback ID:** "settings_menu" → показывает кнопки выбора языка
3. **При нажатии:** "lang_ru" или "lang_uk" 
4. **Обработка:** `handle_language_selection()` сохраняет язык в БД
5. **Результат:** Все сообщения сразу на новом языке
