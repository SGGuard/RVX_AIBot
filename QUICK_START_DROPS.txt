════════════════════════════════════════════════════════════════════

📡 API ENDPOINTS:

GET /get_drops?limit=10&chain=all
    Параметры:
    - limit: кол-во результатов (по умолчанию 10)
    - chain: arbitrum, solana, polygon, ethereum, all
    
GET /get_activities
    Информация об активностях (стейкинг, ланчи, гавернанс)
    
GET /get_trending?limit=10
    Трендовые токены за 24ч
    
GET /get_token_info/bitcoin
    Информация о конкретном токене
