## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Парсинг Markdown

Используется **регулярные выражения** для надежной парсинга:

```python
# **жирный** → <b>жирный</b>
re.sub(r'\*\*([^\*]+?)\*\*', r'<b>\1</b>', text)

# __подчеркнутый__ → <u>подчеркнутый</u>
re.sub(r'__([^_]+?)__', r'<u>\1</u>', text)

# ~~зачеркнутый~~ → <s>зачеркнутый</s>
re.sub(r'~~([^~]+?)~~', r'<s>\1</s>', text)

# `код` → <code>код</code>
re.sub(r'`([^`]+?)`', r'<code>\1</code>', text)

# *курсив* → <i>курсив</i> (безопасно для **)
re.sub(r'(?<!\*)\*([^\*]+?)\*(?!\*)', r'<i>\1</i>', text)
```

### Безопасность

✅ Не создает незакрытые теги  
✅ Корректно обрабатывает множественное форматирование  
✅ Работает с кириллицей и Emoji  
✅ Безопасна для HTML injection (только теги для форматирования)
