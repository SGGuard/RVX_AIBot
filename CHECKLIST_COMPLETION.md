```bash
from i18n import get_text, set_user_language, get_user_language
import asyncio

async def test():
    # Тест 1: Получить текст на русском
    ru_text = await get_text('menu.ask_button', language='ru')
    print(f'🇷🇺 Русский: {ru_text}')
    
    # Тест 2: Получить текст на украинском
    uk_text = await get_text('menu.ask_button', language='uk')
    print(f'🇺🇦 Украинский: {uk_text}')
    
    # Тест 3: Установить язык пользователю
    user_id = 123456
    await set_user_language(user_id, 'uk')
    lang = get_user_language(user_id)
    print(f'✅ Язык пользователя установлен: {lang}')
    
    print('✅ Все тесты пройдены!')

asyncio.run(test())
"
```

## 📈 МАСШТАБИРУЕМОСТЬ

Система легко расширяется:

**Добавить новый язык (например, казахский):**
```
1. Создай locales/kk.json (копия ru.json)
2. Переведи в файле kk.json на казахский
3. В bot.py добавь кнопку: InlineKeyboardButton("🇰🇿 Қазақша", callback_data="lang_kk")
4. В функции set_user_language() добавь проверку "kk"
5. Готово!
```

Система автоматически поддерживает любые языки!
